# RPC 全局错误码

**注：全局错误码可以不在文档中写出**

| 错误码                               | 描述               | 解释                                     |
| ------------------------------------ | ------------------ | ---------------------------------------- |
| UnprocessableEntity.InvalidParameter | 无法处理，非法参数 | 一般的参数格式错误可以使用此状态码       |
| InternalError                        | 服务器错误         | 如事务异常，文件操作异常，网络异常等可用 |
|                                      |                    |                                          |
